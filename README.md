# Contrast Sections iOS App

A simple iOS app that displays a page divided into 5 sections, each with a high-contrast background color.

## Colors Used

The app uses 5 colors chosen for maximum visual contrast:
1. **Red** - RGB(255, 0, 0)
2. **Green** - RGB(0, 255, 0)
3. **Blue** - RGB(0, 0, 255)
4. **Yellow** - RGB(255, 255, 0)
5. **Magenta** - RGB(255, 0, 255)

## Setup

This is a SwiftUI app that requires:
- Xcode 14.0 or later
- iOS 16.0 or later

To run:
1. Open the project in Xcode
2. Select a simulator or device
3. Build and run
